YouTubeDiscordPresence — macOS host

This installs the desktop component (YTDPmac) that powers the YouTubeDiscordPresence browser extension, and registers it as a native-messaging host for your Chromium-family browsers (Chrome, Edge, Brave, Vivaldi, Chromium).

You also need:

  • The YouTubeDiscordPresence extension from the Chrome Web Store.
  • The Discord desktop app running (not the browser version).

The component installs to /Library/Application Support/YouTubeDiscordPresence/. It runs natively on Intel Macs and on Apple Silicon via Rosetta 2.
